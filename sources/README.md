# Primary-source NVC reports

This folder holds the official UCI Beall Center end-of-year NVC reports used to verify the archive's data.

| File | Year(s) | Notes |
|------|---------|-------|
| `NVC-Report-2019-2020.pdf` | 2019-2020 cycle | Covers the 2020 NVC ("New Structure" edition with Baby Box as Grand Prize). |
| `1-NVC-2021-Summary.pdf` | 2021 | Short summary report. |
| `NVC-2022-Report.pptx.pdf` | 2022 | 18-year edition slide deck export. |
| `NVC-2023-Report.pptx.pdf` | 2023 | 19-year edition slide deck export. |
| `NVC-2024-Report.pptx.pdf` | 2024 | 20-year edition slide deck export. |

When these reports disagree with secondary press coverage, the report wins. Each year entry in `data/nvc.json` lists its `primary_source` filename.
